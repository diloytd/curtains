# Калькулятор стоимости штор

Веб-приложение для расчёта заказа штор: параметры проёма, смета (ткань, лента, работа), схема 2D и корзина с печатью. Интерфейс — нейтральный (серый/белый), на базе Material UI.

## Возможности

- ввод ширины, высоты, варианта крепления и коэффициента сборки;
- расчёт стоимости по разбивке: ткань + лента + работа;
- отображение упрощённого 2D-чертежа шторы;
- корзина заказа и печать со страницы.

Подробные требования и **формулы расчёта** — в [`requirements.md`](requirements.md).

## Стек

Vite · TypeScript · React · MUI (Material UI)

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`. Просмотр production-сборки: `npm run preview`.

## Структура репозитория

| Путь | Назначение |
|------|------------|
| `requirements.md` | требования и формулы |
| `src/App.tsx` | корневой layout, корзина, печать |
| `src/theme.ts` | нейтральная тема MUI |
| `src/lib/` | расчёты (`calculations.ts`), типы, цены |
| `src/components/CalculatorForm.tsx` | форма, смета, чертёж |
| `src/components/Cart.tsx` | корзина |
| `src/components/CurtainDrawing.tsx` | 2D-чертёж на canvas |
