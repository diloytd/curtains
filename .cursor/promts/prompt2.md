Создай React компонент CurtainDrawing.tsx, который рисует чертёж шторы.

Входные параметры (props):
- width: number (ширина карниза, метры)
- height: number (высота окна, метры)
- foldRatio: number (коэффициент сборки 1.5-3.0)
- curtainType: 'straight' | 'pleated' | 'roman'

Что рисовать на Canvas 400x300px:
1. Верхняя линия — карниз (от x=50 до x=350, y=40)
2. Штора — прямоугольник:
   - ширина = width * foldRatio, масштаб: 1м = 50px
   - высота = height + 0.2, y от 40 до 40+высота_в_px
3. Для pleated: вертикальные линии внутри шторы (шаг 20px)
4. Для roman: горизонтальные полосы (каждые 50px высоты)
5. Подписи размеров (ширина и высота в метрах)

Ограничения:
- Только чёрно-белый цвет
- Без логотипов, без узоров
- Используй useRef и useEffect для рисования
- Масштабируй под размеры окна (responsive)
